# 🏫 Debark University Student & Admin Portal 

ይህ የደባርቅ ዩኒቨርሲቲ ተማሪዎች እና አስተዳዳሪዎች (Registrar/Admin) በቀላሉ ተግባቦት የሚያደርጉበት፣ ኮርሶችን የሚመዘግቡበት እና ውጤት የሚያስተዳድሩበት የተሟላ የዌብ አፕሊኬሽን ፕሮጀክት ነው።

## ✨ ዋና ዋና ክፍሎች (Features)

### 👨‍🎓 የተማሪዎች ክፍል (Student Portal)
* **Student Registration:** አዳዲስ ተማሪዎች ሙሉ መረጃቸውን እና ፎቶቸውን በማስገባት በቀላሉ የሚመዘገቡበት ፎርም።
* **Secure Login:** በየራሳቸው መለያ ቁጥር (Student ID Number) እና የይለፍ ቃል (Password) በጥንቃቄ መግቢያ ገጽ።
* **Personal Dashboard:** የተማሪውን ጠቅላላ ውጤት (Cumulative GPA)፣ የትምህርት ሁኔታ (Academic Status) እና በአሁኑ ሰዓት እየወሰደ ያላቸውን ኮርሶች በግልጽ የሚያሳይ ገጽ።
* **Course Registration & Grades:** የወሰዷቸውን ኮርሶች እና ውጤታቸውን (Grades) ማየት የሚያስችል አማራጭ።

### 👨‍💼 የአስተዳዳሪዎች ክፍል (Registrar / Admin Panel)
* **Registrar Control Dashboard:** አጠቃላይ የተመዘገቡ ተማሪዎችን ብዛት እና በመጠባበቅ ላይ ያሉ ማመልከቻዎችን (Pending Applications) ማሳያ።
* **Student Management:** የተማሪዎችን ዝርዝር ማየት እና የየራሳቸውን ፕሮፋይል (View Profile) የመቆጣጠር ብቃት።
* **Course & Grade Management:** አዳዲስ ኮርሶችን መጨመር (Add New Courses) እና የተማሪዎችን ውጤት ማስተዳደር።
